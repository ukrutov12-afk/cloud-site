# Cloud Client — сайт

Сайт покупки клиента Cloud: мультиязычность (be / ru / uk / en), рабочая
регистрация/вход, документы (соглашение, конфиденциальность, возврат),
покупка и поддержка.

## Запуск

```bash
npm install
npm start
```

Открыть: http://localhost:3000 (порт можно сменить через `PORT`).

## Что внутри

- **Express + EJS** — серверный рендер страниц.
- **Сессии + bcrypt** — рабочая авторизация (пароли хранятся хешем).
- **i18n** — `locales/{be,ru,uk,en}.json`, переключение языка в шапке (запоминается в сессии).
- **Хранилище** — JSON-файлы в `data/` (users.json, orders.json). Без нативных зависимостей.
- **Документы** — `docs/<lang>/<doc>.html` (ru + en, остальные языки → ru-фолбэк).

## Структура

```
server.js          маршруты, i18n, авторизация, тарифы
db.js              файловое хранилище (пользователи, заказы)
locales/           переводы интерфейса (4 языка)
docs/ru, docs/en   тексты документов
views/             EJS-шаблоны (+ partials, docs/page)
public/css,js      стиль Liquid Glass + скрипты
data/              создаётся автоматически (БД)
```

## Замечания для прод

- Смените `secret` сессии в `server.js` и включите `cookie.secure` за HTTPS.
- Поток оплаты — демонстрационный (заказ создаётся со статусом `pending`).
  Подключите платёжный провайдер в `POST /buy`.
- Для нагрузки замените JSON-хранилище на полноценную БД.
